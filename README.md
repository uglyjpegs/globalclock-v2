# Global Clock v2

A redesigned multi-timezone world clock. Shows real-time clocks for multiple cities. Successor to the original [globalclock](https://github.com/uglyjpegs/globalclock).

## Features

- Real-time updates across all clocks
- Add and remove time zones
- Dark/light theme toggle
- Responsive layout

## Usage

Open `index.html` directly in a browser — no build step or server needed.

## Stack

Vanilla HTML, CSS, and JavaScript. No dependencies.
